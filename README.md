# Google Clone

### 프로젝트 설명

- 바닐라 코딩 사전학습 가이드 제출용
- HTML, CSS만을 이용하여 구글 웹사이트와 비슷한 UI 구현

### 기능 목록

- 입력 및 클릭

  - [x] Search bar 에 마우스를 올릴 시 구글 웹사이트와 비슷한 효과 구현
  - [x] Gmail, Images 및 웹사이트 하단 웹사이트 resources 모두 link 로 구현되어 있기 때문에 클릭 시 실제 구글사이트로 연결
  - [-] 화면 사이즈 조정에 대한 기능은 없음
